# TUTORIAL COMPLETO DE PANDAS EN PYTHON

Este repositorio contiene un tutorial interactivo en formato Jupyter Notebook para aprender a usar Pandas en Python, desde lo más básico hasta funcionalidades avanzadas, con ejemplos prácticos orientados al análisis y manipulación eficaz de datos.

## Contenido del tutorial

- Introducción a Pandas: instalación y conceptos clave
- Estructuras de datos fundamentales: Series y DataFrames
- Indexación, selección y filtrado de datos
- Lectura y escritura de datos en formatos CSV, Excel, JSON, y más
- Limpieza y transformación de datos: manejo de valores faltantes y datos atípicos
- Operaciones avanzadas: agrupaciones, merges, pivotes y más
- Manipulación de fechas y tiempos
- Análisis exploratorio de datos con estadísticas descriptivas y gráficos básicos
- Ejemplos prácticos con conjuntos de datos reales
- Consejos para optimizar el rendimiento y evitar errores comunes

## Cómo empezar

### Instalación en entorno virtual (recomendado)

1. Crea y activa un entorno virtual (ejemplo usando `venv`):

    ```bash
    python -m venv env
    source env/bin/activate  # En Windows usa: env\Scripts\activate
    ```

2. Instala las dependencias:

    ```bash
    pip install -r requirements.txt
    ```

3. Abre el notebook con Jupyter:

    ```bash
    jupyter notebook Tutorial_Pandas.ipynb
    ```

### Ejecución en Google Colab

1. Abre [Google Colab](https://colab.research.google.com/).

2. Sube el archivo `Tutorial_Pandas.ipynb` o ábrelo directamente desde GitHub usando la opción "GitHub" en Colab.

3. Si necesitas instalar dependencias adicionales, ejecuta en una celda:

    ```python
    !pip install pandas numpy matplotlib openpyxl
    ```

4. Ejecuta las celdas del notebook y sigue el tutorial de forma interactiva en la nube.

## Requisitos técnicos

- Python 3.7 o superior
- pandas
- numpy
- matplotlib
- openpyxl (para soporte Excel)
